# Student Grade Management System (C)

This program allows users to input student names and grades, then calculates and displays:

- Individual student grades
- Average grade
- Highest and lowest grades

## Features
- Input validation (1–50 students)
- Array-based data storage
- Loop-based calculations
- Formatted output

## Technologies Used
- C programming language
- Developed using OnlineGDB and local testing
